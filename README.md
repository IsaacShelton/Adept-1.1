# Adept
## A blazing fast low-level language
Adept is designed for performace and simplicity.<br/>
Generally it runs as fast or faster than C code.

# Examples

### Variables
```
def main() int {
    age int = 36
    total = 10 + 3
    return age + total
}
```

### Functions
```
def sum(a int, b int) int {
    return a + b
}

def main() int {
    return sum(13, 8)
}
```
